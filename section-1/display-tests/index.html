Three simple visualizations of Hemlock Data

Source code

These are three simple visualizations based on a dataset about hemlock growth. They were made using p5.js.

Simple line graph

This simple line graph maps the growth and index and raw ring width data points. The scale is the same between the measured ring widths and the growth index. The values of the ring width are mapped from 0-1.5mm to 0-100 pixels. The growth index is mapped from 0-2 to 0-100. The lines are color differentiated but doesn't reveal too much except there appears to be a correlation between the growth index and ring width.

Simple bar graph

This simple bar graph displays two data points chronologically. Going up the bars show the growth index while going down it shows the raw ring width. This graph attempts to show two barcharts and how they correlate, however since the data points are similar it looks like a waveform with the bottom graph simply an inversion of the top graph.

Interactive ring graph

This graph attempts to visually represent the ring width data. By mapping the ring widths to the ellipse width it creates an appearance of tree rings. Rolling over the rings will display the year and ring width measurement. The data is not particularly cleaned so when there are multiple instances of the same measurement the info displays overlapped.
